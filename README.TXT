
A program segítségével sportfogadásra lesz lehetőségünk.
AZ adatok egy XML fájlból kerülnek feldolgozásra. Jelen esetben csupán egy sportágra valósul meg a fogadási lehetőség. A cél, hogy a felhasználó egyértelmű keretek között képes legyen egy saját szelvényt létrehozni, s abból a várható nyereményre tudjon következtetni.
